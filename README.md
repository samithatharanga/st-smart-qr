# 🚀 ST Imagix Smart QR Suite v2.0 — Enterprise Serverless Edition

An ultra-modern, production-ready, and 100% serverless QR Code generator platform engineered for micro-businesses and tech enthusiasts. This upgraded enterprise-grade application runs entirely on the client side, ensuring zero telemetry, absolute data privacy, and premium vector exports.

🔗 **Live Web App:** [https://samithatharanga.github.io/st-smart-qr/](https://samithatharanga.github.io/st-smart-qr/)

---

## ✨ Features & Capabilities (v2.0 Upgrades)

- **📱 Full Mobile-Responsive Engine:** Flawlessly optimized for iOS & Android device viewports.
- **💡 Quick Marketing Templates:** - **URL / General Text Mode:** Standard inputs.
  - **WhatsApp Quick Chat:** Input phone number + pre-filled marketing message templates.
  - **WiFi Auto-Connect Layout:** Instantly structures local SSID credentials into machine-readable network strings.
- **🖼️ "Scan Me" Custom Framing System:** Features interactive Light/Dark toggle matrix renders embedded into an *Elegant Border* or *Minimal* layout context.
- **🎨 Dynamic Localization & Branding:** Inject English or custom local Sinhala typography (e.g., ඇණවුම් කරන්න, සල්ලි ගෙවන්න) inside the custom frames live.
- **📦 Multi-Format Enterprise Export:** - **PNG:** High-resolution raster file.
  - **SVG:** Scalable vector asset optimized for physical banners without pixelation.
  - **PDF:** Instantly compiles an A4 layout configuration ready for desktop print.

---

## 🛠️ Tech Stack & Compliance

- **Framework:** Semantic HTML5 & Modern CSS3 Animation layers.
- **Styling UI:** Tailwind CSS v3 Framework.
- **Core Engines:** HTML5 Canvas API, `html2canvas.js`, `jsPDF.js`, and `qrcode.js`.
- **Privacy Audit:** 100% Serverless. Zero server footprints or tracking scripts.
