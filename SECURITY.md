# Security Policy for ST Imagix Smart QR

## Supported Versions
We currently provide security updates and support for the following versions of the ST Smart QR Suite:

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability
ST Imagix takes security very seriously. Because this application is completely serverless and runs 100% on the client-side (browser), there are no backend databases or API keys exposed. 

However, if you discover any security vulnerabilities or cross-site scripting (XSS) risks, please do not disclose it publicly. 

Instead, please use the **Private Vulnerability Reporting** feature in this GitHub repository to report the issue directly to the ST Imagix development team. We will review and address the issue promptly.
